¡Bienvenido a la aplicación para la asignatura de Sistemas en SpringBoot!

Usa contar/{PALABRA} para contar vocales y consonantes

Usa /addFarmacia?nombre={NOMBRE}&ubicacion={UBICACION}&ventas={VENTAS} para añadir una nueva farmacia al sistema

Usa /listarFarmacias para listar las farmacias del sistema

Usa /mirror/{PALABRA} y te la devuelve en formato espejo

Usa /traduce/{PALABRA/FRASE} para traducir al inglés una palabra o frase
